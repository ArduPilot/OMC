MrSID Decode SDK for Raster Reference Manual

This is the Reference Manual for the MrSID Decode SDK for Raster, containing complete documentation and examples for all the SDK classes and methods.New SDK developers should first review the User Manual, which contains a general introduction to the SDK architecture and the core classes, descriptions of the MrSID and JPEG 2000 imaging features, as well as technical notes on specific topics of interest to SDK developers.
