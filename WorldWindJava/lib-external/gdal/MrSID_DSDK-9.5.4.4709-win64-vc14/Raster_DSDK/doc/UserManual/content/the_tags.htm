The Tags

Certain metadata tags are set by almost all image stages and are known as the “classical” metadata tags:

  • width and height
  • datatype
  • colorspace
  • dynamic range
  • background and nodata pixel values
  • geospatial (x,y) origin, x- and y-resolution, and rotation
  • original file size
  • original filename

The corresponding image tag names can be found in the LTIMetadataTag enum. See "Metadata Tags" for further details.
