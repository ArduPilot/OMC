INSTALLATION:
  No specific installation is required to use the Raster SDK beyond
  copying the SDK contents to your local machine.

  After installing, we suggest building and running the example program
  to assure correct installation and behaviour.

    api examples:  examples/src

  Contents include header files, libraries for Debug and Release, documentation,
  examples and utilities.
   
DOCUMENTATION:
  - User Manual: doc/UserManual/index.htm
  - Reference Manual: doc/ReferenceManual/index.html
  - Additional supporting documentation, additional READMEs, current known
    issues, etc, can be found in the text files in the doc subdirectory.
